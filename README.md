AI Learning Video-like App

An interactive AI-based learning prototype built with Streamlit that explains concepts step-by-step using images, text, AI voice narration, and videos, creating a virtual teacher experience.

Features:

Step-by-step concept explanation

Images for visual learning

AI voice narration (Text-to-Speech)

Embedded YouTube explainer videos

Next/Previous navigation with progress bar

Tech Stack:

Python

Streamlit

gTTS (Google Text-to-Speech)

├── images/
└── README.md

Photosynthesis
